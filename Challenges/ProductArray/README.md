* Product of Array Riddle 

### Challenge
* Given an input of integer array, return the product of all the numbers in the array except the current index it is in.
- Example:
- Input: [1, 2, 3, 4]
- Process flow: [ (2 * 3 * 4), (1 * 3 * 4), (1 * 2 * 4), (1 * 2 * 3) ];
- Output: [24, 12, 8, 6]
